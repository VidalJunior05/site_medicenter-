## 📁 How to download the Project 
<br/>

```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/site_medicenter-.git
    
    # Enter directory
    $ cd medicenter

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
